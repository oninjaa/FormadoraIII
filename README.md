# projeto-contatos

Nome do projeto

projeto-contatos

Objetivo do app

Aplicativo simples para gerenciar contatos — listar usuários de uma API pública e permitir adicionar/consultar contatos via Firebase Realtime Database.

Tecnologias utilizadas

- Ionic + Angular (standalone components)
- Firebase Realtime Database (SDK modular)
- HttpClient (Angular)

Este repositório contém um projeto Ionic básico com páginas para listar e adicionar contatos, um serviço para consumir uma API pública e um serviço para integração com o Firebase Realtime Database.
